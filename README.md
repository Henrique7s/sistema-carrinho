# 🛒 Sistema de Carrinho em C

## 📋 Descrição
Este projeto simula um carrinho de compras onde o usuário pode:
- Inserir produtos
- Informar preços
- Calcular o valor total da compra
- Contar a quantidade de itens

## 🚀 Tecnologias utilizadas
- Linguagem C
- Estruturas de repetição (while)
- Vetores
- Funções
- Ponteiros
